Investigators

All fields with a red asterisk * are required.

 

  1. Select an Organization from the drop down list.
  2. Select a Group.
  3. Select Investigators. Note: To select multiple Investigators, hold down the Ctrl button on your keyboard and click on any names listed.
  4. Click Add Investigators.
  5. Investigators will appear in a box in the right hand side of the page. Use the Status drop down list to select the status of each investigator.
  6. Click Save or Save & Continue to go to the next task page.

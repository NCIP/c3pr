Notifications

All fields with a red asterisk * are required.
 

This page creates e-mail notifications based on specified Threshold criteria. If you do not need to set notifications you can click Save & Continue to go to the next task page.

  1. Click Add Notification.
  2. Enter the Threshold. This is the number of subjects that need to be registered in order to trigger an e-mail notification.
  3. Click Add Email and enter the appropriate e-mail address.
  4. Click the Add Role button and select the appropriate Role from the drop down menu.
  5. Click Save or Save & Continue to go to the next task page.

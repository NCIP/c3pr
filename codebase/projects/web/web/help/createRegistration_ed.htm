Enrollment Details

Warning: If the participant is already registered on this epoch you can move this subject to another epoch of this study with the Manage Registration feature. To navigate to Manage Registration, search for the registration and then click on the registration record. Go to Search for Registrations for more information on how to change the current epoch.

All fields with a red asterisk * are required.

Fill out the Enrollment Details.
 

  1. Enter the Informed Consent Signed Date.
  2. Indicate what the Current Consent Version is. This is the version of the consent document that the subject signed.
  3. Enter the Registration Start Date.
  4. Select the Primary Disease from the drop down list. If the selection is Other, then C3PR provides the user with an addition text field to which it represents a type of disease other than the primary disease for which the patient is being treated could be typed in.
  5. Enter the Primary Disease Site. This is a prepopulated field. Enter the first few letters of the name of the Primary Disease Site and select it from the drop down list that appears.
  6. Select the Payment Method from the drop down list.
  7. Click the Continue to go to the next page (Check Eligibility).

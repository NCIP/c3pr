Personnel

All fields with a red asterisk * are required.

 

  1. Select an Organization. Make a selection from the drop down menu.
  2. Select Research Staff. Note: To select multiple names, hold down the Ctrl button and click on any names listed.
  3. Click Add Research Staff.
  4. Research Staff will appear in a box in the right hand side of the page. Select the appropriate Role and Status from the drop down lists. Go to the Roles section of this guide for an explanation of tasks performed by each role.
  5. Click Save or Save & Continue to go to the next task page.

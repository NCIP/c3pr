IF you are using an existing Tomcat then please append the following line to [TOMCAT_HOME]/conf/catalina.properties

java.security.auth.login.config=[TOMCAT_HOME]/conf/cabig/csm_jaas.config

Note: Replace [TOMCAT_HOME] with the appropriate tomcat directory

Note: Ignore these instructions if you are using the installer to install a new tomcat.

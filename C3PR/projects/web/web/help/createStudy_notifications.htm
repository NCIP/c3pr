Study Notifications

All fields with a red asterisk * are required.
 

You can create e-mail notifications based on specified Threshold criteria that relates to the number that was entered for Target Accrual (the total number of subjects/patients/participants needed for protocol enrollment)  in the Details page.

 

  1. Click Add Notification.
  2. Enter the Threshold. This is the number of subjects/patients/participants that need to be registered in order to trigger an e-mail notification.
    For Example:     If the Target Accrual that was entered on the Details page was 30 and you enter 25 into the Threshold field, an e-mail notification will be sent when the accrual reaches 25.
  3. Click Add Email and enter the appropriate e-mail address.
  4. Click the Add Role button and select the appropriate Role from the drop down list.
  5. Click Save or Save & Continue to go to the next task page.
If you do not need to set notifications you can click Save & Continue to go to the next task page.

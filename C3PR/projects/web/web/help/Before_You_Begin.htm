Before You Begin

The C3PR administrator must enter before end users can enter information and save it in the system.
 

  1. The administrator must first configure C3PR with data for Organizations, Investigators and Research Staff before information can be entered and saved in the system. Go to the Administration tab in the main navigation bar to enter information.

  2. Once Organizations, Investigators and Research Staff  have been entered into the system by the C3PR Administrator, end users can get started by doing the following:

  3.  

    • Learn About User Roles

    • Enter a Study

    • Enter a Subject

    • Register Subjects to a Study
       

If any of the steps above have not been completed by the C3PR system administrator, a study cannot be created or saved in the system. If this is your first time using the C3PR application after it has been installed and configured please check with your C3PR administrator to confirm that all of the necessary information has been entered in C3PR. C3PR administrators can go to the administration section of this guide for more information.
